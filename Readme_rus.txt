Arduino как компьютер с DOS
Ввод и вывод осуществляется через последовательный порт.

Функции:
* Управление файлами (создание, удаление, дописывание)
* Показывает аптайм

Диски:
* A диск на основе RAM (1024 байта)
* B диск на основе ROM
* C диск на основе EEPROM (1024 байта)

Команды:
* a: - сменить текущий диск на A
* b: - сменить текущий диск на B
* c: - сменить текущий диск на C
* ver - версия
* help - краткая справка
* uptime - аптайм
* format - очистить текущий диск
* dir/ls - показать содержимое текущего диска
* show/type/cat - показать содержимое файла
* del/rm - удалить файл
* append - дописать строку в файл
* rewrite - перезаписать файл строкой

Пример работы:
Arduino DOS ver 1.7

C:>uptime
4 s

C:>b:

B:>dir
B: is a ROM disk
Files:
   changelog
   about
   help

Total space: 728 b
Used space: 728 b
Free space: 0 b

B:>
